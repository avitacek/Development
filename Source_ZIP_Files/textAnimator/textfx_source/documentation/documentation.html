Welcome to TextFX

TextFX is easy to use, and you don't need to be a code wizard to use it. For many, the comments written in the index.html file in the download source will be enough to get your animations up and running. But TextFX is also a powerful script that lends itself nicely to more advanced coders with things like:
  1. Optional callbacks with params:
    • onStart
    • onComplete
    • onStartParams
    • onCompleteParams
  2. Automatic overriding: Running a consecutive animation on the same div will automatically kill and clean up the first animation.
  3. Clean up methods: Immediately kill all running animations, a single animation or a selected group of animations.
  4. Start delay for more control on when animations begin.

Methods Available

Special Note:
The jQuery.cjTextFx(); (init function) must always be called before starting your first animation. But it must only be called once.

jQuery.cjTextFx(settings:Object = null);
jQuery.cjTextFx.animate(jQuery-search-term:String, settings:Object = null);
jQuery.cjTextFx.remove(remove-all:Boolean, objects-to-remove:Array = null);


Usage Examples

Special Note:
Params passed into the "animate" function will not be saved as global params. This is why it's a good idea to set some global params (params passed into the init function).

Example #1: Write params to a variable and pass into the init function

<link rel="stylesheet" type="text/css" href="css/cj-text-fx.css" />
<script src="js/cj-text-fx.js" type="text/javascript"></script>

<script type="text/javascript">

jQuery(document).ready(function() {

var textFxSettings = {animation: 3, color: "#FF0000", animationType: "out", speed: 1000};

jQuery.cjTextFx(textFxSettings); // INIT FUNCTION
jQuery.cjTextFx.animate("#div-to-animate");

});

</script>

Example #5: Initialize two animations and then kill them both

<link rel="stylesheet" type="text/css" href="css/cj-text-fx.css" />
<script src="js/cj-text-fx.js" type="text/javascript"></script>

<script type="text/javascript">

jQuery(document).ready(function() {

jQuery.cjTextFx(); // INIT FUNCTION
jQuery.cjTextFx.animate("#div-to-animate", {animation: 3, color: "#FF0000", speed: 1000});
jQuery.cjTextFx.animate("#another-div-to-animate", {animation: 3, startDelay: 3000});

jQuery.cjTextFx.remove(true); // STOPS AND REMOVES ALL ANIMATIONS

});

</script>


Example #6: Animate two divs and kill only the second

<link rel="stylesheet" type="text/css" href="css/cj-text-fx.css" />

<script type="text/javascript">

jQuery(document).ready(function() {

jQuery.cjTextFx.animate("#div-to-animate", {animation: 3, color: "#FF0000", speed: 1000});
jQuery.cjTextFx.animate("#another-div-to-animate", {animation: 7, color: "#000000", startDelay: 3000});

jQuery.cjTextFx.remove(false, ["#another-div-to-animate"]); // ONLY KILLS THE SECOND ANIMATION

});

</script>


Example #7: Animate a div with a callback and call back params

<link rel="stylesheet" type="text/css" href="css/cj-text-fx.css" />
<script src="js/cj-text-fx.js" type="text/javascript"></script>

<script type="text/javascript">

jQuery(document).ready(function() {

var textFxSettings = {animation: 3, color: "#FF0000", animationType: "out", speed: 1000};

jQuery.cjTextFx(textFxSettings); // INIT FUNCTION
jQuery.cjTextFx.animate("#div-to-animate", {onComplete: myFunction, onCompleteParams: ["hello", "world"]);

});

// CALLBACK
function myFunction(ar) {
alert(ar[0] + " " + ar[1]); // OUTPUT: hello world
}

</script>

Optional Settings: (param, default value)

The animation number, Options are 1-15

animation: 1

Options are "in" and "out"

animationType: "in"

Start with the last letter, true/false

backwards: false

The easing type, case sensitive, all options shown on this page are included: http://gsgd.co.uk/sandbox/jquery/easing/

easing: "easeOutQuint"

The animation speed in milliseconds

speed: 1000

The delay between each letter animation in milliseconds

sequenceDelay: 100

The start delay for your animation in milliseconds

startDelay: 0

If the letters are broken apart, this is the offset x value in pixels

offsetX: 100

If the letters are broken apart, this is the offset y value in pixels

offsetY: 50

The font color

color: "#000"

If the text should be hyperlinked, true/false

linked: true

If the text is hyperlinked, enter the url here, otherwise you can enter null

hyperlink: "http://www.google.com"

If the text is hyperlinked, enter the link target here, options are "_blank" or "_parent", otherwise you can enter null

linkTarget: "_parent"

A callback function once the animation has finished, enter null for no callback

onComplete: onCompleteFunction

An array of parameters for the onComplete callback, enter null for no callback params, keep in mind that the
callback will receive a single param (the array)

onCompleteParams: ["string", 5, anotherParam]

A function to fire when the animation starts, enter null for no onStart function

onStart: onStartFunction

An array of params to be passed into the onStart function, enter null for no onStart params, keep in mind that the
callback will receive a single param (the array)

onStartParams: ["string", 5, anotherParam]

Setting this to "false" is useful when changing the text value of a div before running a consecutive animation on the same div.
See the "demo.html" file in the preview folder for a good example of making use of this parameter. Options are true/false

restoreHTML: true



Additional Notes

  • It is possible to use @font-face. Unfortunately using @font-face can produce incorrect width and height values when animations run as soon as the jQuery(document).ready() function fires. Google has a nice @font-face loader script but it is not a cross-browser solution. Because of this, if you choose to use @font-face fonts, it is best to add a "startDelay" to your first animation. I've found that about a second seems to do the trick.

  • TextFX is not compatible with Cufon

  • HTML 5 Doctype is not needed for the animation to work
